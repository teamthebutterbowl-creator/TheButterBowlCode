# Food Ordering Platform

MERN stack food ordering platform.

## Folders

| Folder              | Description                       |
|---------------------|-----------------------------------|
| `userSide/Backend`  | User-facing API (Express + MongoDB) |
| `frontend`          | React landing page (The Butter Bowl) |

## Quick start

```bash
cd userSide/Backend
npm install
copy .env.example .env
npm run dev
```

Setup details: [userSide/Backend/README.md](userSide/Backend/README.md)

### Frontend

```bash
cd frontend
npm install
npm run dev
```
