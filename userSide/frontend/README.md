# The Butter Bowl — Frontend

Mobile-first React landing page for the food ordering platform.

## Tech stack

- React (Vite)
- Tailwind CSS
- Google Fonts: Playfair Display + Inter

## Run locally

```bash
cd frontend
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Color palette

| Token   | Hex       |
|---------|-----------|
| Primary | `#3E7B35` |
| Gold    | `#E6A817` |
| Cream   | `#F0EDE4` |
| Dark    | `#1A1A1A` |

Defined in `tailwind.config.js`.
