# Restaurant SaaS Admin Panel

Production-ready React 19 admin dashboard scaffold built with Vite.

## Features
- Responsive dashboard layout
- Fixed sidebar and sticky header
- Dashboard, Menu, Orders, Offers, Home CMS, Settings pages
- Reusable components and modern admin UI
- Pure CSS styling with CSS variables

## Run locally
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Open browser at `http://localhost:5173`
